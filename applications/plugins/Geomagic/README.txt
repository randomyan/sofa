To install this plugin you need:

- to register on the Geomagic website
- to download and install the Geomagic driver (following the documentation and the instructions of INSTALL)
- activate the plugin in cmake-gui
- compile it
- enjoy the power of the Geomagic haptic interfaces with SOFA and give a try to the example scene (examples/DemoGeomagic.scn)

And let us know about your Geomagic simulations!
